# paper-write

Write publication-ready content under these rules:

- no unsupported claims
- explicit citation anchors
- clear limitation section
- reviewer-facing clarity over rhetorical style

## Matsuo-quality supplement

When drafting English papers, apply this additional checklist inspired by
Yutaka Matsuo's guidance:

- End introduction with 2-3 explicit contribution bullets.
- Explain community utility ("how others build on this"), not just novelty.
- Keep narrative grounded in shared community problems and prior work context.
- Ensure figures/tables can communicate the story with captions alone.
- Include at least one clear reproducibility anchor (protocol/algorithm detail).

If the draft misses any item, report a short gap list before finalizing.
